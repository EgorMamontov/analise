# analise
Скриптн для автоматизации промтинга на базе исходного кода.

# Комбинация клавиш по управлению выбора файлов:

| Команда | Описание                                       |
| ------- | ---------------------------------------------- |
| N       | инвертировать файлы в каталоге                 |
| N*      | рекурсивное инвертирование                     |
| +N*     | рекурсивное выделение                          |
| -N*     | рекурсивное снятие                             |
| N-E     | работа с конкретными типами файлов             |
| N-E*    | Рекурсивная работа с конкретными типами файлов |
где:
N - номер файла или директории
E - Номер типа файла
